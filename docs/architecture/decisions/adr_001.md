# ADR-001: No Training in Pre-Pilot

Status: Accepted

## Decision

The pre-pilot uses retrieval only. The model is a local GGUF-format model (3B class, e.g., Qwen or Llama 3) run via llama.cpp. No fine-tuning or training of any kind will be attempted during this phase.

## Why

Training requires curated datasets, compute infrastructure, and evaluation tooling not yet warranted at this scale. A 3B local model used for retrieval-augmented generation over a small, well-structured corpus is sufficient for the pre-pilot goals and avoids premature complexity.

## Implications

- All knowledge the model uses comes from retrieved corpus content, not model weights.
- Model must be run locally; no API dependency.
- If a hosted or larger model is needed for a specific task, that is a later decision (new ADR required).

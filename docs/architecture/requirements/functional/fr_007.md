# FR-007: Grounded Answering

## Requirement

The system shall generate answers using only retrieved material and return the supporting evidence alongside every answer.

## Details

- The model is instructed to answer solely from the supplied context (retrieved chunks/claims). See ADR-001 and ADR-007.
- Every answer response includes the list of chunks and/or claims used as evidence.
- If the retrieved context does not contain sufficient information to answer, the system must return a "cannot answer" response rather than hallucinate.
- The model prompt must explicitly prohibit the model from drawing on knowledge outside the provided context.

## Example

```txt
Question:  "What did the Covenant of Varkaar believe about oath-breaking?"

Answer:    "The Covenant of Varkaar regarded oath-breaking as a capital offense,
            enforced under Covenant law."

Evidence:
  - claim_0091 (fact): "The Covenant regarded oath-breaking as a capital offense."
    source: chunk_0042, "Saskan Canon — Varkaar Covenant"
```

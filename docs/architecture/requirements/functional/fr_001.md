# FR-001: Source Ingestion

## Requirement

The system shall register lore source documents with a stable identifier before any processing begins.

## Details

Each document record must include:

| Field | Description |
| --- | --- |
| `document_id` | stable unique identifier (assigned at registration) |
| `title` | human-readable name |
| `source_path` | path to the source file under `data/lore_texts/` |
| `scope` | lore domain (e.g., `"varkaar"`) |
| `ingested_at` | timestamp of registration |

Only documents within the active scope (Covenant of Varkaar for the pilot) are
eligible for ingestion. See ADR-006.

## Example

```txt
document_id: doc_001
title:       "Saskan Canon — Varkaar Covenant"
source_path: data/lore_texts/SaskanCanon-VarkaarCovenant.pdf
scope:       varkaar
ingested_at: 2026-03-29T10:00:00
```
